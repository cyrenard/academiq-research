# AcademiQ Research v1.1.7

This release promotes the experimental Minimal UI work into the main app build and ships the latest local-first research workflow fixes.

## Highlights

- Minimal UI skin is now bundled as the main active skin name (`minimal`).
- Existing legacy experimental skin preferences migrate automatically.
- ISBN-based book lookup is supported from the DOI/URL input flow.
- Browser Capture can detect ISBNs and submit book captures.
- Workspace notebooks, folder-style library organization, Word import, PDF download, and capture metadata paths include the latest fixes.

## Validation

- Automated tests are expected to pass before release.
- Windows installer and blockmap are produced by the release build.

No intentional breaking changes.
